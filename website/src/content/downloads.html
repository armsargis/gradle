Gradle nightly

For snapshots we use our Artifactory snapshot repository. The recommended way to start using the snapshot: First, find the interesting distribution in the repository, then use it to configure the wrapper. Alternatively, you can manually download the snapshot distribution from the repository and install it locally.

Note: old snapshots are periodically deleted. It's generally a good idea to copy the snapshot to some other location, and refer to it from there.
