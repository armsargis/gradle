Development Mailing Lists

The Gradle development team uses the Gradle Developers mailing list to discuss issues and ideas for the development of Gradle itself. Everyone is welcome to join the list and participate in the discussions. Please note however that this list doesn't provide support for using Gradle. The Gradle Forums site is the appropriate channel to use for getting help with Gradle.

There is also a (read only) Gradle SCM list that receives commit and CI build notifications.
