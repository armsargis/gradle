The Gradle Community

Getting help

The first place to start is with the documentation. In particular, the user guide contains a lot of reference material. Also, keep in mind that Gradle build scripts are simply Groovy code that drives the Gradle API, so you might find the API documentation useful.

If you don't find your answer in the documentation, you might try the Gradle Forums.

Contributing to Gradle

Gradle can always benefit from community contributions.

If you are thinking about getting involved with the development or promotion of Gradle please see the page on contributing.
